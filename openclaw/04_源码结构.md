## 3. 数据库脚本位置

```
sql/
├── init.sql                             # 建表脚本（8张表）
│   ├── 用户表 user
│   ├── 分类表 category
│   ├── 书籍表 book
│   ├── 收货地址表 address
│   ├── 购物车表 cart
│   ├── 订单表 order
│   ├── 订单明细表 order_item
│   └── 评论表 review
│
├── data.sql                             # 初始化数据
│   ├── 分类数据（10-20条）
│   ├── 测试书籍数据（30-50条）
│   ├── 测试用户数据（2-3个）
│   └── 管理员账号
│
└── test-data.sql                        # 测试数据
    └── 完整的业务场景测试数据
```

## 4. 关键配置文件位置

| 文件 | 路径 | 说明 |
|------|------|------|
| Maven配置 | `pom.xml` | 项目根目录 |
| 主配置 | `src/main/resources/application.yml` | 数据库、端口等 |
| 日志配置 | `src/main/resources/logback-spring.xml` | 日志输出 |
| MyBatis映射 | `src/main/resources/mapper/*.xml` | SQL映射文件 |
| Web配置 | `WEB-INF/web.xml` | Servlet配置 |
| 前端入口 | `WEB-INF/views/layouts/main.jsp` | 页面布局 |

## 5. 包命名规范

| 包名 | 职责 | 示例 |
|------|------|------|
| `com.example.shixun` | 根包 | 项目主包 |
| `com.example.shixun.controller` | 控制层 | 用户请求入口 |
| `com.example.shixun.service` | 业务层 | 业务逻辑处理 |
| `com.example.shixun.mapper` | 数据访问层 | 数据库操作 |
| `com.example.shixun.entity` | 实体类 | 数据库表映射 |
| `com.example.shixun.dto` | 数据传输对象 | 层间数据传递 |
| `com.example.shixun.vo` | 视图对象 | 返回前端数据 |
| `com.example.shixun.common` | 通用类 | 工具类、常量 |
| `com.example.shixun.exception` | 异常处理 | 统一异常处理 |
| `com.example.shixun.config` | 配置类 | 框架配置 |
