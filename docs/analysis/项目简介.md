# 网上书店系统 - 项目简介

## 项目概述

开发一个基于 Web 的网上书店系统，为用户提供书籍浏览、购买、订单管理等功能。

## 主要功能模块

### 1. 用户模块
- 用户注册 / 登录 / 注销
- 个人信息管理（修改密码、收货地址）
- 会员等级与折扣

### 2. 商品模块
- 书籍分类管理（文学、科技、儿童等）
- 书籍信息管理（名称、作者、出版社、价格、库存、简介、目录）
- 书籍搜索与推荐

### 3. 购物模块
- 购物车管理（添加、删除、修改数量）
- 下单与结算
- 折扣应用（会员价、促销活动）

### 4. 订单模块
- 订单创建与保存
- 订单查询与取消
- 订单状态跟踪（待付款、待发货、已发货、已完成）

### 5. 后台管理模块
- 书籍管理（增删改查）
- 订单管理（处理、发货）
- 用户管理
- 统计数据（销售额、热门书籍）

## 技术栈

| 层级 | 技术 |
|------|------|
| 后端 | Spring Boot 2.x + MyBatis |
| 前端 | HTML/CSS/JS（layui 或 Vue） |
| 数据库 | MySQL 8.0 |
| 构建工具 | Maven |
| 版本控制 | Git |

## 数据库核心表

1. **user** - 用户表
2. **book** - 书籍表
3. **category** - 分类表
4. **cart** - 购物车表
5. **order** - 订单表
6. **order_item** - 订单明细表
