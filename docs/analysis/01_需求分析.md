# 网上书店系统需求分析文档

---

## 1. 系统功能模块列表

### 1.1 用户模块
- 用户注册（邮箱/手机号）
- 用户登录（账号密码）
- 个人信息管理（查看/修改）
- 收货地址管理（增删改查）
- 密码修改与找回

### 1.2 书籍模块
- 书籍搜索（标题/作者/ISBN）
- 书籍分类浏览
- 书籍详情查看（封面、作者、简介、价格、库存）
- 书籍评分与评论
- 热门/新书/推荐列表

### 1.3 购物车模块
- 加入购物车
- 修改购买数量
- 删除购物车商品
- 批量选择结算
- 购物车商品列表

### 1.4 订单模块
- 提交订单
- 订单支付（模拟）
- 订单列表查看
- 订单详情
- 取消订单
- 确认收货
- 订单状态流转

### 1.5 后台管理模块
- 书籍管理（增删改查）
- 分类管理（增删改查）
- 用户管理（查看/禁用）
- 订单管理（查看/处理）
- 评论管理（审核/删除）
- 数据统计看板

---

## 2. 用户角色与权限说明

| 角色 | 权限范围 |
|------|---------|
| 游客 | 浏览书籍、搜索、注册 |
| 普通用户 | 购物车、订单、评论、个人信息管理 |
| 管理员 | 后台管理全功能 |

---

## 3. 业务流程图

### 3.1 注册登录流程

```
游客 → 注册页面 → 填写信息 → 验证 → 注册成功 → 登录 → 首页
                                              ↓
                    登录失败 ← ← ← ← ← ← ← ← ← ← ←
```

### 3.2 购物流程

```
浏览/搜索书籍 → 查看详情 → 加入购物车 → 去结算 → 确认订单 → 支付（模拟） → 订单完成
```

### 3.3 订单流程

```
提交订单 → 待支付 → 已支付 → 待发货 → 已发货 → 待收货 → 已完成
                    ↓          ↓         ↓
                取消订单    退款      退货退款
```

## 4. 功能详细描述

### 4.1 用户模块详细描述

| 功能 | 描述 | 输入 | 输出 |
|------|------|------|------|
| 注册 | 用户填写信息创建账号 | 用户名、密码、邮箱、手机 | 注册成功/失败提示 |
| 登录 | 账号密码验证登录 | 账号、密码 | 登录成功跳转首页 |
| 个人信息管理 | 查看和修改个人信息 | 修改后的信息 | 更新成功提示 |
| 收货地址 | 管理收货地址 | 地址信息 | 地址列表/增删改结果 |

### 4.2 书籍模块详细描述

| 功能 | 描述 | 输入 | 输出 |
|------|------|------|------|
| 搜索 | 按关键词搜索书籍 | 搜索词 | 书籍列表 |
| 分类浏览 | 按分类筛选书籍 | 分类ID | 分类下书籍列表 |
| 详情查看 | 查看书籍完整信息 | 书籍ID | 书籍详情页 |
| 评论 | 对书籍进行评分和评论 | 评分、内容 | 评论提交结果 |

### 4.3 购物车模块详细描述

| 功能 | 描述 | 输入 | 输出 |
|------|------|------|------|
| 加入购物车 | 将书籍加入购物车 | 书籍ID、数量 | 加入成功提示 |
| 修改数量 | 调整购买数量 | 书籍ID、数量 | 更新结果 |
| 删除 | 移除购物车商品 | 书籍ID | 删除结果 |
| 结算 | 选择商品去结算 | 选中商品ID | 跳转订单确认页 |

### 4.4 订单模块详细描述

| 功能 | 描述 | 输入 | 输出 |
|------|------|------|------|
| 提交订单 | 创建订单 | 收货地址、支付方式 | 订单号 |
| 支付 | 完成支付（模拟） | 订单号 | 支付结果 |
| 查看订单 | 查看个人订单列表 | 筛选条件 | 订单列表 |
| 取消订单 | 取消未支付订单 | 订单号 | 取消结果 |

### 4.5 后台管理模块详细描述

| 功能 | 描述 | 权限 |
|------|------|------|
| 书籍管理 | 添加/编辑/删除/上下架书籍 | 管理员 |
| 分类管理 | 添加/编辑/删除书籍分类 | 管理员 |
| 用户管理 | 查看用户列表/禁用账号 | 管理员 |
| 订单管理 | 查看订单/修改订单状态 | 管理员 |
| 评论管理 | 查看/删除违规评论 | 管理员 |
| 数据统计 | 销售额/订单量/用户量统计 | 管理员 |

## 5. 非功能性需求

### 5.1 性能需求
- 页面响应时间 < 3秒
- 支持100并发用户

### 5.2 安全需求
- 用户密码加密存储（MD5/SHA-256）
- SQL注入防护
- XSS防护

### 5.3 可靠性需求
- 数据定期备份
- 异常操作日志记录

### 5.4 兼容性需求
- 主流浏览器兼容（Chrome、Firefox、Edge）
- 响应式布局支持移动端
