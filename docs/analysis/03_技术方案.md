## 1. 技术栈选型

### 1.1 后端技术

| 技术 | 版本 | 说明 |
|------|------|------|
| **Java** | 17 | LTS版本，稳定可靠 |
| **Spring Boot** | 3.2.x | 主流后端框架 |
| **MyBatis-Plus** | 3.5.x | MyBatis增强，简化CRUD |
| **MySQL** | 8.0 | 关系型数据库 |
| **Druid** | 1.2.x | 数据库连接池 |
| **Lombok** | 1.18.x | 简化实体类代码 |
| **Jackson** | 2.15.x | JSON序列化 |

### 1.2 前端技术

| 技术 | 版本 | 说明 |
|------|------|------|
| **Layui** | 2.8.x | 轻量级UI框架，适合管理后台 |
| **原生HTML/CSS/JS** | - | 前台页面 |

> **注意**：如果追求更好的前端体验，可以考虑：
> - **Vue 3 + Element Plus**（更现代，但学习成本较高）
> - **Thymeleaf**（后端渲染，适合快速开发）

### 1.3 项目管理

| 技术 | 说明 |
|------|------|
| **Maven** | 依赖管理和项目构建 |
| **Git** | 版本控制 |

## 3. 各层职责说明

### 3.1 Controller层（控制层）

- **职责**：接收请求、参数校验、调用Service、返回响应
- **命名规范**：`XxxController`
- **常用注解**：
  - `@RestController` / `@Controller`
  - `@RequestMapping`
  - `@GetMapping` / `@PostMapping`
  - `@RequestBody` / `@RequestParam`

### 3.2 Service层（业务层）

- **职责**：处理业务逻辑、事务管理
- **命名规范**：`XxxService`（接口）+ `XxxServiceImpl`（实现）
- **常用注解**：
  - `@Service`
  - `@Transactional`

### 3.3 Mapper层（数据访问层）

- **职责**：数据库CRUD操作
- **命名规范**：`XxxMapper`
- **常用注解**：
  - `@Mapper`
  - `@Select` / `@Insert` / `@Update` / `@Delete`
- **或者**使用XML映射文件

## 5. 项目搭建步骤

### 5.1 环境准备

1. 安装 JDK 17+
2. 安装 MySQL 8.0+
3. 安装 IDEA（Ultimate版支持Spring Boot）
4. 安装 Maven

### 5.2 创建项目

1. **IDEA创建Spring Boot项目**
   - File → New → Project → Spring Initializr
   - 选择 Web 依赖

2. **添加依赖到 pom.xml**

3. **创建包结构**

4. **配置 application.yml**

5. **创建启动类**

```java
@SpringBootApplication
@MapperScan("com.example.shixun.mapper")
public class ShixunApplication {
    public static void main(String[] args) {
        SpringApplication.run(ShixunApplication.class, args);
    }
}
```

### 5.3 数据库初始化

1. 创建数据库 `bookstore`
2. 执行 `sql/init.sql` 建表
3. 执行 `sql/data.sql` 初始化数据

### 5.4 运行测试

1. 启动 `ShixunApplication`
2. 访问 `http://localhost:8080`
3. 确认无报错

## 6. 统一返回格式

```java
@Data
public class Result<T> {
    private Integer code;      // 状态码：200成功，400失败
    private String message;   // 消息
    private T data;           // 数据
    
    public static <T> Result<T> success(T data) {
        Result<T> r = new Result<>();
        r.code = 200;
        r.message = "success";
        r.data = data;
        return r;
    }
    
    public static <T> Result<T> error(String message) {
        Result<T> r = new Result<>();
        r.code = 400;
        r.message = message;
        return r;
    }
}
```

## 7. 跨域配置

```java
@Configuration
public class CorsConfig implements WebMvcConfigurer {
    @Override
    public void addCorsMappings(CorsRegistry registry) {
        registry.addMapping("/**")
                .allowedOriginPatterns("*")
                .allowedMethods("GET", "POST", "PUT", "DELETE")
                .allowedHeaders("*")
                .allowCredentials(true)
                .maxAge(3600);
    }
}
```
