## 五、技术方案

### 5.1 pom.xml 核心依赖

- Spring Boot 3.2.5
- MyBatis-Plus 3.5.6
- Druid 1.2.23
- MySQL Connector 8.0
- Lombok
- Jakarta Servlet
- Layui 2.8.x

### 5.2 application.yml 配置

```yaml
server:
  port: 8080
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/bookstore
    username: root
    password: root
mybatis-plus:
  configuration:
    map-underscore-to-camel-case: true
```

## 九、注意事项

### 9.1 GitHub 仓库权限
- 需要在 GitHub Settings → Collaborators 添加组员
- 仓库目前是 Private，必须添加 collaborators 才能 push

### 9.2 数据库初始化
- MySQL 中创建 `bookstore` 数据库
- 执行 `sql/init.sql` 建表和初始数据
- 默认管理员：用户名 `admin`，密码 `123456`（需确认）

### 9.3 常见问题
- **push 失败**：检查网络，确保已添加为 collaborator
- **合并冲突**：先 `git pull` 拉取最新代码
- **IDEA 导入**：File → Open → 选择项目根目录
